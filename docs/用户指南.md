# YunShan 使用指南（零基础版）

> 这份文档假设你**完全不懂量化**。读完你能明白：这系统是干嘛的、怎么装起来、
> 怎么用、以及最重要的——**回测结果里那一堆指标到底怎么看**。

---

## 一、这个系统是干嘛的？

一句话：**它能帮你用历史数据，验证一个"炒股套路"在过去到底赚不赚钱。**

举个例子。你听人说一种炒股方法："5 日均线上穿 20 日均线就买，下穿就卖"。
听起来好像有道理，但**它过去十年真照这么做，到底是赚是亏？赚多少？中间会不会
一度亏到让你睡不着？** 凭感觉是答不上来的。

YunShan 做的事就是：

1. **自动下载**某只股票过去的真实行情数据；
2. 让电脑**严格按这个套路**，从过去某天开始一天天模拟买卖；
3. 算出**如果你当年真这么操作，今天账户会变成什么样**——画成一条收益曲线；
4. 再用一堆指标告诉你：**赚了多少、最惨亏过多少、值不值、比啥都不干强不强**。

这个过程行话叫 **「回测」（Backtesting）**——拿历史"回头测试"一个策略。

### 它能帮你做什么

- **检验一个想法靠不靠谱**：别人吹的"神奇指标"，拉过去数据一跑就知道是不是吹牛。
- **对比不同套路**：双均线和 MACD 哪个在这只股票上更稳？跑一跑比一比。
- **找合适的参数**：均线到底用 5/20 还是 10/30 好？让系统自动帮你试一遍。
- **建立盘感和敬畏心**：你会直观看到，大部分"看起来很美"的策略其实跑不赢"买了就拿着"。

### ⚠️ 它不能帮你做什么（很重要）

- **不能预测未来**。过去赚钱不代表将来赚钱，这是铁律。回测好看的策略，实盘亏钱很常见。
- **不接实盘、不会帮你下单买股票**。它纯粹是个"研究和学习工具"。
- **不构成任何投资建议**。这是个人学习项目，亏钱了不能赖它 🙂。

## 二、几个最基础的概念（看懂指标前先扫盲）

| 行话 | 大白话 |
|---|---|
| **策略** | 一套固定的买卖规则。比如"金叉买、死叉卖"。 |
| **信号** | 策略在每一天给出的指令：该持有(1)还是空仓(0)。 |
| **回测** | 拿历史数据，按策略规则一天天模拟交易，看最终结果。 |
| **净值 / 权益曲线** | 假设你一开始有 10 万，按策略操作后，账户钱数随时间变化的那条线。 |
| **基准 (买入持有)** | 对照组：一开始就把钱全买入、然后啥也不干拿到底。**策略必须跑赢它才算有用。** |
| **回撤** | 账户从一个高点往下跌的幅度。最大回撤就是历史上跌得最惨的那次。 |
| **复权** | 把分红、送股的影响抹平，让价格连续可比。系统默认用"前复权"，你不用管。 |

## 三、怎么安装 / 启动

### 0. 前提

电脑上要有 **Python**（项目自带 `.venv` 虚拟环境）和 **Node.js**（前端用）。
本说明里的命令都在**项目根目录** `YunShan/` 下执行。

### 1. 装依赖（只需做一次）

```bash
# Python 后端依赖
.venv/bin/pip install -r requirements.txt

# 前端依赖
cd frontend && npm install && cd ..
```

### 2. 启动（每次使用）

要开**两个终端窗口**，都在项目根目录：

```bash
# 终端 1 —— 启动后端（数据和计算都在这）
.venv/bin/uvicorn backend.main:app --reload

# 终端 2 —— 启动前端（你看的网页）
cd frontend && npm run dev
```

然后浏览器打开 **http://localhost:5173** ，就是操作界面了。
关闭：在两个终端里分别按 `Ctrl + C`。

### 3. 偷懒版（只想快速看看，不想开两个终端）

```bash
.venv/bin/streamlit run app.py
```

这是个简化版网页，功能差不多，适合快速试。但正式用建议还是上面的前后端版。

### 常见问题：拉数据报错 `ProxyError`？

如果换个没用过的股票代码时报这个错，多半是你电脑开了**科学上网代理**（如 Clash）。
行情数据来自国内服务器，本该直连。系统已自动处理大部分情况；万一还报错，
在启动后端的终端里先执行一次：

```bash
unset http_proxy https_proxy all_proxy HTTP_PROXY HTTPS_PROXY ALL_PROXY
```

再重新启动后端即可。（注：之前能跑的股票是因为数据已缓存在本地、没走网络。）

## 四、怎么用（操作流程）

打开 http://localhost:5173 后，左侧是控制面板：

1. **股票代码**：填 6 位代码，如 `000001`（平安银行）、`600519`（贵州茅台）。
2. **开始 / 结束日期**：回测的时间范围，比如 2022-01-01 到 2023-12-31。
3. **策略**：从下拉里选一个（4 个内置策略见下一节）。
4. **参数**：选了策略后会自动出现它的参数（如均线天数），可以改。
5. 点 **「运行回测」** → 右侧出现指标卡 + 两张图。

进阶玩法：

- **网格寻优**：选好"寻优目标指标"（比如夏普），点「网格寻优」，系统会把这个策略
  的参数各种组合都跑一遍，列出排名，还画一张热力图——**哪组参数最好一眼看到**。
- **走动检验**：点「走动检验」，把历史切成几段，"前一段挑出的最好参数，拿到后一段
  去验证"。如果前一段神、后一段拉胯，说明那参数是**过拟合**（只是碰巧拟合了历史，
  没有真本事）。这是判断策略可不可信的高级武器。

## 五、内置的 4 个策略（大白话版）

| 策略 | 思路 | 适合的行情 |
|---|---|---|
| **ma_cross（双均线交叉）** | 短期均线升到长期均线上方就持有，跌破就空仓。追涨杀跌的趋势跟随。 | 单边上涨/下跌的趋势行情 |
| **macd_trend（MACD 趋势）** | 用 MACD 指标判断多头还是空头，多头时持有。也是趋势跟随。 | 趋势行情 |
| **rsi_reversal（RSI 反转）** | 跌过头（超卖）时抄底买入，涨过头（超买）时卖出。逆势抄底摸顶。 | 来回震荡的行情 |
| **bollinger_revert（布林带回归）** | 价格跌破"下轨"时认为超跌、买入，回到中间时卖出。也是逆势。 | 震荡行情 |

> 没有哪个策略永远最好。趋势策略在震荡行情会被反复打脸，反转策略在单边行情会
> 抄底抄在半山腰。**这正是你要用回测去观察、体会的东西。**

## 六、★ 怎么看那一堆指标 ★（本文档重点）

跑完回测，上方会出现 8 张指标卡。每张卡有三个东西帮你看懂：

- **彩色徽章**：🟢不错 / 🟡一般 / 🔴偏差 / ⚪参考 —— 不用懂数字，看颜色就知好坏。
- **大白话点评**：卡片下方一句话，直接说人话。
- **问号 `?`**：鼠标悬停上去，有更详细的解释。

下面把每个指标讲透。**记住一个总原则：永远不要只看收益，要收益和风险一起看。**

### 收益类——"赚了多少"

**① 年化收益率**
把整段时间的总收益，折算成"平均每年涨百分之几"。
- 怎么看：越高越好。但单看它没意义，必须结合下面的风险和基准。
- 🟢 >10% ｜ 🟡 0~10% ｜ 🔴 亏损

**② 超额收益（vs 买入持有）⭐ 最该看的一个**
= 你的策略年化 −「一开始全买、拿到底」的年化。
- 怎么看：**只有大于 0，才说明你这套择时操作真有用**；如果 ≤0，那你忙活半天
  择时买卖，还不如一开始买了就躺着不动。**很多花哨策略都死在这一条。**
- 🟢 多赚 >3% ｜ 🟡 多赚 0~3% ｜ 🔴 还不如躺着

### 风险类——"可能亏多惨"

**③ 最大回撤**
历史上账户从某个最高点，最惨跌到过多少（是个负数）。
- 怎么看：越接近 0 越好。−10% 比 −40% 安全得多。它代表你**心理和钱包扛不扛得住**
  ——回撤 −50% 意味着要再涨 100% 才回本。
- 🟢 比 −10% 浅 ｜ 🟡 −10%~−30% ｜ 🔴 比 −30% 深

### 性价比类——"每冒一份险，换来多少回报"

这几个是"风险调整后收益"，专业人士最看重。它们回答一个关键问题：
**这收益是靠玩命冒险堆出来的，还是真有本事？**

**④ 夏普比率**
每承担"一份波动（涨跌起伏）"，能换来多少超额回报。
- 怎么看：粗略标准 `<0.5` 偏差，`0.5~1.5` 一般，`>1.5` 不错，`<0` 是亏钱。
- 🟢 >1.5 ｜ 🟡 0.5~1.5 ｜ 🔴 <0.5

**⑤ Sortino 比率**
和夏普几乎一样，但**只把"下跌"算作风险**（上涨的波动不算坏事）。更贴近人的真实感受。
- 怎么看：同夏普，越高越好，通常比夏普略高一点。

**⑥ Calmar 比率**
= 年化收益 ÷ 最大回撤。回答"我冒着最惨那次大跌的风险，每年能换多少收益"。
- 怎么看：越高越好，`>1` 算不错。它对"会不会突然爆个大亏"特别敏感。

### 交易质量类——"交易做得怎么样"（看好坏要小心）

**⑦ 交易胜率**（标 ⚪参考，因为高不一定好）
你完成的每一笔"买入→卖出"里，赚钱的占多少。
- 怎么看：**高不一定好，低不一定差！** 趋势策略经常胜率只有 40%，但靠少数几笔
  大赚，覆盖掉多数小亏，照样总体赚钱。所以**别拿胜率当主要标准**，要结合总收益看。

**⑧ 交易笔数**（标 ⚪参考）
回测期间一共完整买卖了几次。
- 怎么看：主要用来判断**样本够不够**。如果只有 2、3 笔，那上面的胜率、夏普全都
  没什么说服力（运气成分太大）。系统会在笔数 <5 时标黄提醒你。

## 七、一套"看回测"的标准动作

拿到结果别懵，按这个顺序看，3 步就能下判断：

1. **先看「超额收益」** → 是不是 >0？不是的话，这策略基本可以否了（跑不赢躺平）。
2. **再看「最大回撤」** → 这个亏损幅度你扛得住吗？扛不住，收益再高也跟你没关系。
3. **然后看「夏普 / Calmar」** → 确认这收益不是靠疯狂冒险换来的。
4. **最后瞄一眼「交易笔数」** → 太少（<5）的话，前面的结论都打个问号，多换几段时间再验证。

再加一个**心法**：跑出来好看别激动，去点一下「走动检验」，
看看"样本外"还灵不灵——能扛住走动检验的，才稍微可信一点。

## 八、最后再强调一遍

- 回测**只能告诉你过去**，不能预测未来。
- 这是**学习和研究工具**，帮你理解市场、训练判断力，**不是印钞机**。
- 任何结果都**不构成投资建议**。真要投钱，风险自负、量力而行。
